## AaaS MVP: Modelo de Dados Simplificado para Blueprints

Para o MVP, o modelo de dados para um blueprint será simplificado e incluirá os seguintes atributos:

-   **ID**: Identificador único do blueprint.
-   **Nome**: Nome legível do blueprint (ex: 'Microservice-Template').
-   **Descrição**: Breve descrição do que o blueprint representa.
-   **Versão**: String de versão (ex: '1.0.0').
-   **Caminho do Artefato**: Onde o arquivo principal do blueprint está armazenado.
-   **Metadados**: Um campo JSON opcional para metadados adicionais (tags, autor, etc.).

Este modelo suportará o gerenciamento básico de blueprints para o escopo do MVP.
